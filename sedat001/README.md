# About Code #

- This is the reusable jar including all the custom validators.

## Tech Stack ##

- Java 8
- Springboot
- Kafka Streams API

## Setup Guide ##

1. Install Dependencies : `` mvnw install ``

2. Start Stream : `` mvnw spring-boot:run ``
