# About Code #

- This is the car microservice which is a part of myfleet.

## Tech Stack ##

- Java 8
- Springboot
- JPA (Spring Data)
- H2 DB
- Model Mapper

## Setup Guide ##

1. Install Dependencies : `` mvnw install ``

2. Run the Code : `` mvnw spring-boot:run ``
